# Trooper Tracker

This is a small play aid for [DANGER CLOSE](https://dicegoblingames.itch.io/danger-close-tactical-skirmish-ttrpg) by Dice Gobin Games.

Based on the demo rules, it currently allows simple management of troopers including:

- Equipment
- Tactical Positioning
- Status

It also allows to store and manage multiple troopers.

This is a personal project for personal use and learning.

This project includes code generated by AI; Google Gemini and Claude.
